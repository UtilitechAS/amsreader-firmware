# Kamstrup payloads

> **`gh73-1/2/3` are decryptable** — the project's first encrypted meter (issue #73). The EK/AK were emailed and are recovered in the keystore
> (`AMS_TEST_KEY_EM20200710_*`); they decrypt + authenticate to valid DLMS.
